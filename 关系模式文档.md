## 关系模式定义

>目前所有关系模式均符合BCNF范式

### User

| 属性名            | 数据类型 | 备注                      |
| ----------------- | -------- | ------------------------- |
| user_id           | int      | 主码、主属性              |
| username          | varchar  | 主属性                    |
| password          | varvhar  | -                         |
| identity          | int      | 0：管理员 1：家教 2：学生 |
| registration_date | date     | -                         |

### Student

| 属性名  | 数据类型 | 备注         |
| ------- | -------- | ------------ |
| user_id | int      | 主码、主属性 |
| age     | int      | -            |
| gender  | int      | 0：男 1：女  |
| contact | varchar  | 主属性       |
| email   | email    | 主属性       |
